Приложение было создано для анализа прайс-листов конкурента и отслеживания его показателей

===ЛОГИКА===

1) Раз в сутки приложение заходит на почтовый ящик куда ежедневно присылаются необходимые прайс-листы. Находит нужные письма по отправителю, теме письма, названию файла.

2) Затем парсит данные из прайс-листов в базу данных. Часть данных заносятся напрямую в БД, остальные рассчитываются на основе занесенных данных.

==================================

Приложение имеет веб-интерфейс чтобы пользователи могли просматривать данные.
